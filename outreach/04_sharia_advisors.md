## 📋 Что нужно подготовить для шариа-советника

### Минимальный пакет документов:
1. **Описание бизнес-модели** (1–2 страницы) включая:
   - Модель ujrah (сервисный сбор 1–2.5%) — объяснить почему это не riba
   - Процесс скоринга (как определяются получатели)
   - Smart contract механизм
   - Отсутствие speculative token

2. **Pitch deck** (уже есть: deck/AM-Network-Deck.html)

3. **Список вопросов для шариа-ревью:**
   - Является ли ujrah модель допустимой для Zakat платформы?
   - Является ли AI-скоринг допустимым способом определения mustahiq (получателей)?
   - Совместимы ли smart contracts с принципами ислама?
   - Возможна ли анонимная передача Zakat (это важно для многих доноров)?

## ⏱ Реалистичный timeline шариа-сертификации

| Этап | Время |
|------|-------|
| Первичный контакт с советником | Неделя 1 |
| Бесплатная консультация | Недели 1–2 |
| Engagement letter + оплата | Неделя 2–3 |
| Sharia review бизнес-модели | 2–4 недели |
| Первое мнение (non-binding) | Месяц 1–2 |
| Официальный Sharia Certificate | Месяц 3–6 |
| Полноценный Sharia Board | Месяц 6–12 |

**Совет:** Даже "Sharia review in progress" с именем известного учёного значительно поднимет доверие при инвесторских встречах.

## 💰 Бюджет на Sharia Advisory

| Услуга | Стоимость |

| Первичная консультация | $0–1,000 |
| Первичный Sharia audit (бизнес-модель) | $3,000–15,000 |
