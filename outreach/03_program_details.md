**Статус:** ✅ Уже подан. Дедлайн 2 августа 2026. Ответ ожидается в течение 3 месяцев.

**Что дают:**

**Процесс:** Подача → Скрининг (июнь–ноябрь) → Интервью → Решение → Онбординг (февраль 2027)

**Контакт:** hub71.com/program/hub71-plus-digital-assets | hub71@hub71.com

## Как финансирование работает шаг за шагом

```
ГРАНТ (XRPL, Gitcoin, Algorand, Google.org)
├── Подаёшь заявку
├── Получаешь деньги (безвозвратно)
├── Отчитываешься о результатах
└── Не теряешь долю компании ← ЛУЧШИЙ ВАРИАНТ СЕЙЧАС

АКСЕЛЕРАТОР (Hub71, DIFC, QFTH, Flat6Labs)
├── Подаёшь заявку
├── Проходишь интервью
├── Зачисляют в программу (2-4 месяца)
├── Получаешь деньги + менторство
├── Отдаёшь долю (0–10% зависит от программы)
└── Demo Day → привлекаешь следующий раунд

ВЕНЧУРНОЕ ФИНАНСИРОВАНИЕ (Shorooq, 500 Global, YC)
├── Готовишь pitch deck
├── Отправляешь инвестору
├── Серия встреч + due diligence
├── Term sheet (условия сделки)
├── Отдаёшь долю (6–25%)
└── Получаешь деньги на счёт юр. лица
```

## Нужно ли юридическое лицо?

| Действие | Юр. лицо нужно? |
|---------|----------------|
| Подать заявку в акселератор | ❌ Нет |
| Участвовать в акселераторе | 🟡 Желательно |
| Получить деньги от акселератора | ✅ Да |
| Подать на грант ($<25K) | ❌ Нет |
| Получить грант деньгами | ✅ Да |
| Pitch VC | ❌ Нет |
| Закрыть раунд с VC | ✅ Да |

**Рекомендация:** Зарегистрируйся в **Malaysia Labuan** как только будет первая встреча с инвесторами. Стоимость: ~$1,500–2,000. Срок: 2–4 недели.
