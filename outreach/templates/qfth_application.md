## Шаг 3 — Когда откроется форма Wave 9 (октябрь 2026)

### Описание компании:
```
AM Network — первая блокчейн-платформа верификации Закята с AI-скорингом 
получателей (0-100 по 15 критериям), смарт-контрактами Base blockchain и 
оракульной сетью (имамы + НКО). Модель: уджра 1-2.5% — строгое соответствие шариату.
```

### Проблема:
```
$600 млрд потенциал Закята в год (Всемирный Банк + IRTI), <$25 млрд распределяется.
Три фундаментальные проблемы: нет прозрачности, нет верификации получателей, 
нет цифровой инфраструктуры. Катар как центр исламских финансов идеально подходит 
для создания такой инфраструктуры.
```

### Связь с исламским финансами:
```
AM Network — 100% исламский финтех:
• Закят: один из пяти столпов ислама
• Уджра: признана допустимой многими шариа-учёными
• Ноль riba: ноль процентов, ноль спекулятивных токенов
• Соответствие асхаб: обслуживаем все 8 категорий получателей Закята
```

### Рынок:
```
• $600 млрд Закят (Всемирный Банк + IRTI 2016)
• $341 млрд исламский финтех к 2029, CAGR 11.5% (GIFT Report, QFC 2025)
• $6 трлн рынок исламских финансов (LSEG 2024)
• 2 млрд мусульман (Pew Research 2024)
```
