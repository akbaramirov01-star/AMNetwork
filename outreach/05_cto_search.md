# Поиск CTO — Полный гайд

> CTO — самый важный шаг сейчас. Без него заблокированы: смарт-контракты, AI API, все инвесторы класса A.

---

## Профиль идеального CTO

### Hard skills (обязательно хотя бы 3 из 5):
- [ ] Solidity / EVM smart contracts (Base, Ethereum)
- [ ] Web3.js / Ethers.js / Viem
- [ ] Python + ML (для AI scoring API)
- [ ] Node.js или FastAPI бэкенд
- [ ] Понимание blockchain архитектуры (UTXO, EVM, gas, oracles)

### Soft skills (важно):
- Понимание или интерес к исламским финансам
- Готовность к equity-based (не зарплата, а доля)
- Доступность для удалённой работы (мы distributed)
- Опыт запуска продукта (не только code monkey)

### Что предлагаем:
- **5–20% equity** в зависимости от опыта и вклада
- Роль CTO + Co-Founder
- Решаешь реальную проблему 2 миллиардов мусульман
- Реальный рынок: $600B Zakat + $6T Islamic Finance
- Уже готовы: сайт на 8 языках, смарт-контракт прототип, Apply форма, AI scoring

## Где искать — по приоритету

### 1. LinkedIn (лучший канал)

**Поиск:**
- Ключевые слова: `Solidity developer` + `Web3` + `Islamic finance`
- Или: `blockchain developer` + `fintech` + `MENA`
- Или: `smart contracts` + `Zakat` / `Islamic`
- Фильтр по локации: UAE, Malaysia, Indonesia, UK, Germany, Turkey

**Что делать:**
1. Создай привлекательный пост (шаблон ниже)
2. Напрямую пиши разработчикам — персонализированное сообщение
3. Опубликуй в группах: "Islamic Finance Professionals", "Web3 MENA", "Muslim Entrepreneurs"

## Красные флаги при выборе CTO

❌ Хочет только зарплату (не equity) — не партнёр
❌ Нет опыта продуктовой разработки, только кодинг
❌ Не понимает исламские финансы и не хочет учиться
❌ Нет портфолио живых проектов
❌ Требует >30% equity сразу без contribution

## Зелёные флаги

✅ Работал в blockchain/Web3 продуктах которые запустились
✅ Знает или интересуется Islamic finance / Zakat
✅ Задаёт вопросы про продукт, не только технологию
✅ Готов начать со small test task (paid или proof of interest)
✅ Имеет LinkedIn с реальными проектами

## CTO Equity Framework

| Профиль | Equity |
|---------|--------|
| Студент / junior (< 2 лет) | 3–7% |
| Mid-level developer (2–5 лет) | 7–12% |
| Senior с реальным Web3 опытом | 12–18% |
| Опытный CTO с запущенными продуктами | 15–25% |

**Совет:** Начни с 4-часового тестового задания. Платить $100–300 за него. Это покажет реальный уровень до того как давать equity.
